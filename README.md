# Установка
## 1. Зависимости
Чтобы запустить проект, сначала убедитесь, что у вас установлены **Docker** и **Docker-Compose**  
  
## 2. Переменные среды
Чтобы проект заработал, нужно настроить следующие переменные среды внутри файла *.env*:  
  
**BOT_API** - API Телеграм бота  
**AI_AUTH_KEY** - Ключ авторизации Гигачата  
  
**POSTGRES_USER** - Суперпользователь PostgreSQL (рандомное значение)  
**POSTGRES_PASSWORD** - Пароль суперпользователя PostgreSQL (рандомное значение)  
**POSTGRES_DB** - База данных PostgreSQL (рандомное значение)  
**POSTGRES_RO_USER** - Пользователь с правами чтения PostgreSQL (рандомное значение)  
**POSTGRES_RO_PASSWORD** - Пароль пользователя с правами чтения PostgreSQL (рандомное значение)  
  
## 3. Запуск
Запустите проект в докере.
### Windows
```
.\deploy.bat
```
### Unix
```
./deploy.sh
```
